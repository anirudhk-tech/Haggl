## üèÜ PROBLEM STATEMENTS (Pick ONE)

### Statement One: Prolonged Coordination
Create an agentic system capable of performing intricate, multi-step workflows that last **hours or days**, utilizing MongoDB as the context engine, while enduring failures, restarts, and modifications to tasks.

**Key Challenges:**
- Execute tool calls across long time periods
- Retain reasoning state
- Recover from single failures
- Ensure task consistency in multi-step tasks

### Statement Two: Multi-Agent Collaboration
Develop a multi-agent system in which specialized agents explore, assign tasks, and communicate with one another, using MongoDB to organize and oversee contexts.

**Key Challenges:**
- How agents convey their skills
- Identify suitable peers for sub-tasks
- Share context effectively within token limits
- Perform intricate tasks from successful collaborations

### Statement Three: Adaptive Retrieval
Create an agentic retrieval system that actively fetches from various resources (databases, search indices, multimodal sources, websites, etc.): modifying query approaches, altering chunking, reordering results based on input.

**Key Challenges:**
- Create agentic and adaptive retrieval
- Improve system over time
- Perform reasoning across various documents and sources

### Statement Four: Agentic Payments and Negotiation
Create agents capable of finding, negotiating, and purchasing services automatically through the **x402 protocol**.

**Key Challenges:**
- Oversee budget across various services
- Perform good-faith negotiation of price
- Evaluate service worth
- Manage payment errors smoothly
- Review all financial choices in MongoDB

**Special Track:**
- **Coinbase Developer Track** - Agentic Commerce on MongoDB: CDP Wallets + x402-Driven Tool Chaining ($2,000 USDC prize)
